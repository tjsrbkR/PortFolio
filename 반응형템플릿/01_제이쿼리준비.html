2.슬라이더 영역
사이트 아이덴티티 이미지가 표현되는 영역 = 키비쥬얼 이미지 #. 가로 전체 너비 사용
3-1. 디스플레이 영역
타이틀, 내용이 들어감 모바일 + 태블릿 = > 세로배치, PC => 가로배치
3-2. 프로모션 영역 아이콘, 타이틀, 간단한 내용, 광고를 위한 영역 흥미를 위해 아이콘등을 배치 => 클릭을 유도 모바일 => 세로배치, 태블릿 + PC => 가로배치
3-3. 최근작업 영역 => 홍보상품, 포트폴리오 최근에 작업한 작업물을 보여주기 위한 영역 최대한 잘 보이도록 퀄리티 높은 이미지 사용 모바일 => 세로배치, 태블릿 + PC => 가로배치
3-4. 블로그 영역 => 게시판, 공지사항, 뉴스 등등 최근에 작성한 글들을 보여주는 영역 모바일 => 세로배치, 태블릿 + PC => 가로배치
3-5. 연락처 영역 웹 사이트 관리자에게 연락하기 위한 영역 모바일 => 세로배치, 태블릿 + PC => 가로배치
